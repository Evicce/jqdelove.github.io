多态存在的条件

1、类的继承

提到多态就必须要讲一下继承的概念，当一个类继承于另一个类时，该类则是前一个类的子类，而被继承的类就称为这个子类的父类。子类继承父类可以得到父类的全部属性和方法（构造方法除外），即子类对象可以调用父类的未被重写的所有方法。而当没有写出该类继承于哪一个类时，默认继承Object类，即Object类是所有类的父类。

2、方法的重写

指的是子类重写父类的方法，当父类方法不能满足子类的实现时，需要重写父类的方法。
